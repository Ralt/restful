The restful dictionary


[Standard class]
acceptor



[Standard class]
collection



[Generic function]
create-resource resource post-data => result


This function creates a new resource.


[Method]
create-resource (resource resource) post-data => result



[Generic function]
delete-item storage identifier => result


Deletes a single item in the storage.


[Method]
delete-item (storage memory-storage) identifier => result



[Generic function]
delete-resource resource => result


This function deletes an existing resource.


[Method]
delete-resource (resource resource) => result



[Generic function]
get-item storage identifier => result


Gets a single item in the storage.


[Method]
get-item (storage memory-storage) identifier => result



[Generic function]
get-items storage => result


Returns all the items in the storage.


[Method]
get-items (storage memory-storage) => result



[Generic function]
has-permission resource method => result


This function determines if the request has permission to hit the resource.


[Method]
has-permission (resource resource) method => result



[Generic function]
load-resource resource => result


This function loads a resource based on its identifier.


[Method]
load-resource (resource resource) => result

[Generic function]
patch-resource resource post-data => result


This function patches an existing resource.


[Method]
patch-resource (resource resource) post-data => result



[Generic function]
replace-resource resource post-data => result


This function replaces a resource.


[Method]
replace-resource (resource resource) post-data => result

resource



[Generic function]
resource-action resource => result


This function lets you handle actions on the resource. The :identifier slot lets you know which action is called.


[Method]
resource-action (resource resource) => result
